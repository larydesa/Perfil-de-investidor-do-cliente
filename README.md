# Perfil de investidor do cliente
Verificando se o perfil de um cliente é conservador, moderado ou arrojado em um investimento

Nesse programa é criado uma carteira de investimento onde o cliente tem a opção de conhecer o perfil de investidor dele para aplicar o seu dinheiro da maneira correta, além disso, ele tem a opção de suporte ao cliente, seja por um agente virtual ou um especialista
